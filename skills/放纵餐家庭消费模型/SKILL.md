# Skill：放纵餐家庭消费规划

---

## 一、Skill说明

用于为有长期饮食控制需求的家庭用户提供周期性放纵餐的结构化规划与建议，平衡心理调节需求与健康边界，避免失控和极端补偿。

---

## 二、输入（Runtime Input）

- 用户原始意图输入（自然语言描述放纵餐需求）
- 用户私域数据（家庭画像 / 偏好 / 历史行为）

## 四、执行逻辑（Core Logic）

### Step1：约束解析
- 从用户输入中提取：
  - 功能约束：进食顺序、份量控制、频率规划、时间建议、前置策略
  - 数量约束：每周1-2次，热量≤1.5-2倍
  - 质量约束：避免高糖+高脂，识别风险人群
  - 交付约束：结构化计划输出
  - 情感约束：缓解焦虑，正向鼓励
  - 预算约束：可选

### Step2：能力筛选
规则：
- 从【能力清单】中筛选能力
- 仅保留满足约束的能力
- 必须覆盖所有「必填约束」
- 禁止新增能力

### Step3：能力组合
- 去除冗余能力
- 确保能力链完整：
  1. 饮食需求识别 → 2. 放纵餐策略匹配 → 3. 热量边界计算 → 4. 风险人群评估 → 5. 饮食方案生成 → 6. 心理健康引导

### Step4：结果校验
- 校验是否覆盖所有必填约束
- 若未覆盖 → 从能力清单中补齐
