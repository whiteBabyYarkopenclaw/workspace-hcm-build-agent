# Intent定义

意图领域：
餐饮美食

意图名称：
家庭饮食需求

意图描述：
针对不同家庭类型，拆解其饮食健康目标、约束条件与需求优先级，沉淀为可落地执行的家庭饮食方案需求模型

Intent六维约束：

功能：
拆解不同家庭类型的饮食健康目标、约束条件与需求优先级，生成可落地执行的家庭饮食方案

数量：
可同时处理多个家庭成员的需求

质量：
基于中国居民膳食指南、营养素参考摄入量等专业标准

交付：
结构化家庭饮食方案，包含目标、原则、规划、建议

情感：
家庭和谐、仪式感

成本：
优先使用常见食材，考虑时间与预算约束
