# Intent定义

意图领域：
医疗健康

意图名称：
外伤应急处理

意图描述：
外伤应急处理用于在受伤发生后的第一时间，对出血、伤口、骨折等情况进行快速处置，以控制伤情、预防感染并为后续专业医疗救治争取时间。

Intent六维约束：

功能：
对出血、伤口、骨折等情况进行快速处置，控制伤情、预防感染、为后续专业医疗救治争取时间

数量：
支持单人或多人多处伤口

质量：
基于标准急救指南，确保处理方法科学安全

交付：
步骤式指导，清晰易懂

情感：
安心、不慌张

成本：
优先使用家庭常备急救物品
