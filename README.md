## 核心概念

**放纵餐（Cheat Meal）** 是指在长期饮食控制过程中，周期性安排的一餐相对自由的饮食行为。

### 核心目的
1. **心理调节** - 缓解饮食压力
2. **提高可持续性** - 让饮食控制更长久
3. **代谢刺激** - 部分人群适用

### 关键原则
- 放纵 ≠ 失控
- 允许高热量，但不无限制
- 需要结构性规划，而非冲动消费

## 专家知识体系

### 1. 放纵餐优化策略

| 策略维度 | 具体内容 |
|----------|----------|
| 进食顺序 | 蛋白质 → 蔬菜 → 高热量食物（降低血糖波动） |
| 份量控制 | 70%-80%饱，避免连续加餐 |
| 频率控制 | 每周1-2次，避免连续多餐 |
| 时间选择 | 运动后/白天优于深夜 |
| 前置策略 | 放纵餐前摄入高蛋白，减少暴食风险 |

### 2. 常见误区

| 误区 | 说明 |
|------|------|
| 把放纵餐当"放纵日" | 导致热量严重超标 |
| 情绪驱动进食 | 容易形成暴食习惯 |
| 极端补偿（第二天不吃） | 破坏代谢稳定 |

### 3. 健康边界

- 单次热量不超过日常的 **1.5~2倍**
- 不连续两餐放纵
- 避免高糖+高脂组合（如奶茶+炸鸡）

### 4. 风险人群

- 血糖异常人群
- 暴食倾向人群
- 严格医疗饮食人群
